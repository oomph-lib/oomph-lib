Refineable collapsible channel mesh. The mesh is derived from the {\ttfamily \doxylink{classoomph_1_1SimpleRectangularQuadMesh}{Simple\+Rectangular\+Quad\+Mesh}} so it\textquotesingle{}s node and element numbering scheme is the same as in that mesh. Only the boundaries are numbered differently to allow the easy identification of the "{}collapsible"{} segment. Boundary coordinates are set up for all nodes located on boundary 3 (the collapsible segment). The curvilinear ("{}collapsible"{}) segment is defined by a {\ttfamily Geom\+Object}.
