Adaptative version of the \doxylink{classoomph_1_1QuarterTubeMesh}{Quarter\+Tube\+Mesh} base mesh. The domain is specified by the Geom\+Object that identifies boundary 3. 

The mesh boundaries are numbered as follows\+:
\begin{DoxyItemize}
\item Boundary 0\+: "{}\+Inflow"{} cross section; located along the line parametrised by $ \xi_0 =  \xi_0^{lo} $ on the geometric object that specifies the wall.
\item Boundary 1\+: Plane x=0
\item Boundary 2\+: Plane y=0
\item Boundary 3\+: The curved wall
\item Boundary 4\+: "{}\+Outflow"{} cross section; located along the line parametrised by $ \xi_0 =  \xi_0^{hi} $ on the geometric object that specifies the wall. 
\end{DoxyItemize}
