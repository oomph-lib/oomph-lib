class oomph\+::\+Vector$<$ \+\_\+\+Tp $>$}

A slight extension to the standard template vector class so that we can include \char`\"{}graceful\char`\"{} array range checks if the R\+A\+N\+G\+E\+\_\+\+C\+H\+E\+C\+K\+I\+NG flag is set.
