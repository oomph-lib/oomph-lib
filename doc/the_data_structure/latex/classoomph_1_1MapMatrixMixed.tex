\mbox{\hyperlink{classoomph_1_1MapMatrixMixed}{Map\+Matrix\+Mixed}} is a generalised, S\+T\+L-\/map-\/based, sparse(ish) matrix class with mixed indices

The matrix is indexed by indices of type K\+E\+Y\+\_\+\+T\+Y\+P\+E\+\_\+\+R\+OW and K\+E\+Y\+\_\+\+T\+Y\+P\+E\+\_\+\+C\+OL and has entries of type V\+A\+L\+U\+E\+\_\+\+T\+Y\+PE.

Careful\+: If a zero entry is referenced then it is created in memory. Therefore this isn\textquotesingle{}t really a practical sparse matrix scheme. Do not loop over `all' possible indices as even looking at them will inflate the matrix until it occupies as much space as a full one -- use (modification of) output routine to retrieve all nonzero entries.

However, this is not a serious restriction, as the main purpose of this class is to allow non-\/integer indices.
