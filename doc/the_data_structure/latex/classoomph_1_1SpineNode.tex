The assumption is that each \mbox{\hyperlink{classoomph_1_1Node}{Node}} lies at a fixed fraction on a single spine (although more complex behaviour could be included by adding more variables to the spine). In general, more complex node updating should be handled by the classes implemented for algebraic node updates.
