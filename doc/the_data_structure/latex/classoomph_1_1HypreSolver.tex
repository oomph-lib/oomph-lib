\doxysubsection{Detailed Description}
An \mbox{\hyperlink{classoomph_1_1LinearSolver}{Linear\+Solver}} class using the suite of Hypre solvers to allow

Boomer\+A\+MG (A\+MG), \mbox{\hyperlink{classoomph_1_1CG}{CG}}, \mbox{\hyperlink{classoomph_1_1GMRES}{G\+M\+R\+ES}} or \mbox{\hyperlink{classoomph_1_1BiCGStab}{Bi\+C\+G\+Stab}}

to be used for Linear\+Solver\+::solve(...) or Linear\+Solver\+::resolve(...).

The Krylov subspace solvers (\mbox{\hyperlink{classoomph_1_1CG}{CG}}, \mbox{\hyperlink{classoomph_1_1GMRES}{G\+M\+R\+ES}} and \mbox{\hyperlink{classoomph_1_1BiCGStab}{Bi\+C\+G\+Stab}}) may be preconditioned using\+: \begin{DoxyVerb}    BoomerAMG, Euclid or ParaSails
\end{DoxyVerb}


By default \mbox{\hyperlink{classoomph_1_1GMRES}{G\+M\+R\+ES}} without preconditioning is used.
