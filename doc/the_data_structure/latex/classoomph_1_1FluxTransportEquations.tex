Base class for the flux transport equations templated by the dimension DIM. The equations that are solved are. 

\[ \frac{\partial u_{i}}{\partial t} + \frac{\partial}{\partial x_{j}} \left(F_{ij}(u_{k})\right), \] where $ F_{ij} $ is a matrix of flux components.
