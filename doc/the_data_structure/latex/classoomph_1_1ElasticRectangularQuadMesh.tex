/////////////////////////////////////////////////////////////////////// /////////////////////////////////////////////////////////////////////// 

Elastic quad mesh with functionality to attach traction elements to the specified boundaries. We \char`\"{}upgrade\char`\"{} the \mbox{\hyperlink{classoomph_1_1RectangularQuadMesh}{Rectangular\+Quad\+Mesh}} to become an \mbox{\hyperlink{classoomph_1_1SolidMesh}{Solid\+Mesh}} and equate the Eulerian and Lagrangian coordinates, thus making the domain represented by the mesh the stress-\/free configuration.
