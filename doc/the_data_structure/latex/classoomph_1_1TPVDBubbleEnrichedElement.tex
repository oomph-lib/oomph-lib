\subsubsection*{template$<$unsigned DIM, unsigned NNODE\+\_\+1D$>$\newline
class oomph\+::\+TPVDBubble\+Enriched\+Element$<$ DIM, NNODE\+\_\+1\+D $>$}

///////////////////////////////////////////////////////////////////// ///////////////////////////////////////////////////////////////////// 

An Element that solves the solid mechanics equations, based on the principle of virtual displacements in Cartesian coordinates, using Solid\+TBubble\+Enriched\+Elements for the interpolation of the variable positions. These elements are typically required when using pseudo-\/elasticity to move internal mesh nodes and TCrouzeix\+Raviart\+Fluid elements.
