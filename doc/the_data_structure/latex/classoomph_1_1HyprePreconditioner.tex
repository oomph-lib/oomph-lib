\doxysubsection{Detailed Description}
///////////////////////////////////////////////////////////////////// ///////////////////////////////////////////////////////////////////// 

An \mbox{\hyperlink{classoomph_1_1Preconditioner}{Preconditioner}} class using the suite of Hypre preconditioners to allow \begin{DoxyVerb} BoomerAMG (Algebraic Multi Grid),
 Euclid (ILU) or
 ParaSails (Approximate inverse)
\end{DoxyVerb}
 to be used for Preconditioner\+::preconditioner\+\_\+solve(...). By default Boomer\+AMG is used.
