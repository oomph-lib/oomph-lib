\subsubsection*{template$<$unsigned DIM$>$\newline
class oomph\+::\+Impose\+Flux\+For\+Womersley\+Element$<$ DIM $>$}

//////////////////////////////////////////////////////////////////////// //////////////////////////////////////////////////////////////////////// 

Element to impose volume flux through collection of Womersley elements, in exchange for treating the pressure gradient as an unknown. The pressure gradient is created (as a single-\/valued \mbox{\hyperlink{classoomph_1_1Data}{Data}} item) in the constructor for this element which also takes a pointer to the \mbox{\hyperlink{classoomph_1_1Mesh}{Mesh}} containing the Womersley elements whose total flux is being controlled. While doing this we tell them that their pressure gradient is now an unknown and must be treated as external \mbox{\hyperlink{classoomph_1_1Data}{Data}}.
