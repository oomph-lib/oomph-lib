\doxysubsection{Detailed Description}
An interface class to the suite of Hypre solvers and preconditioners to allow use of\+:

Boomer\+A\+MG (A\+MG), \mbox{\hyperlink{classoomph_1_1CG}{CG}}, \mbox{\hyperlink{classoomph_1_1GMRES}{G\+M\+R\+ES}} or \mbox{\hyperlink{classoomph_1_1BiCGStab}{Bi\+C\+G\+Stab}}, Euclid (I\+LU) or Para\+Sails (Approximate inverse)
