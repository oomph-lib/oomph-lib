The FGMRES method, i.\+e. the flexible variant of the \doxylink{classoomph_1_1GMRES}{GMRES} method which allows for nonconstant preconditioners \mbox{[}see Saad Y, "{}\+Iterative methods for sparse linear systems"{}, p.\+287\mbox{]}. Note, FGMRES can only cater to right preconditioning; if the user tries to switch to left preconditioning they will be notified of this.
