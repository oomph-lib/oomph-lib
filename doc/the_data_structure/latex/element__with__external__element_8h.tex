\begin{DoxyCompactList}\small\item\em This is a base class for all elements that require external sources (e.\+g. FSI, multi-\/domain problems such as Helmholtz, multi-\/mesh Boussinesq convection, etc.). It provides storage for the source element and corresponding local coordinate at each integration point, and allows use of locate\+\_\+zeta to obtain such source elements. In addition separate storage is allocated for all field data in the external elements and all geometric data that can affect the field data in the external elements. Generic finite difference routines are provided to calculate entries in the Jacobian from the data of the external elements. \end{DoxyCompactList}\end{DoxyCompactItemize}
