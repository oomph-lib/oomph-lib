\subsubsection*{template$<$class T, class MATRIX\+\_\+\+TYPE$>$\newline
class oomph\+::\+Sparse\+Matrix$<$ T, MATRIX\+\_\+\+TYPE $>$}
//////////////////////////////////////////////////////////////// //////////////////////////////////////////////////////////////// 

Class for sparse matrices, that store only the non-\/zero values in a linear array in memory. The details of the array indexing vary depending on the storage scheme used. The MATRIX\+\_\+\+TYPE template parameter for use in the curious recursive template pattern is included and passed directly to the base \mbox{\hyperlink{classoomph_1_1Matrix}{Matrix}} class.
