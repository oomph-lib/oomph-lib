\doxysection{oomph\+::Continuation\+Storage\+Scheme Class Reference}
\hypertarget{classoomph_1_1ContinuationStorageScheme}{}\label{classoomph_1_1ContinuationStorageScheme}\index{oomph::ContinuationStorageScheme@{oomph::ContinuationStorageScheme}}


Generalised\+Timestepper used to store the arclength derivatives and pervious solutions required in continuation problems. The data is stored as auxilliary data in the (fake) \doxylink{classoomph_1_1TimeStepper}{Time\+Stepper} so that spatial adaptivity will be handled automatically through our standard mechanisms. The adopted storage scheme is that the continuation derivatives will be stored at the first auxilliary value and the previous value will be the second auixilliary value.

\doxysubsection{Detailed Description}
Generalised\+Timestepper used to store the arclength derivatives and pervious solutions required in continuation problems. The data is stored as auxilliary data in the (fake) \doxylink{classoomph_1_1TimeStepper}{Time\+Stepper} so that spatial adaptivity will be handled automatically through our standard mechanisms. The adopted storage scheme is that the continuation derivatives will be stored at the first auxilliary value and the previous value will be the second auixilliary value.
