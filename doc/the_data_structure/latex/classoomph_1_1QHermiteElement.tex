class oomph\+::\+Q\+Hermite\+Element$<$ D\+I\+M $>$}

General \mbox{\hyperlink{classoomph_1_1QHermiteElement}{Q\+Hermite\+Element}} class. Local coordinates are not assumed to be aligned with the global coordinates so the Jacobian of the mapping between local and global coordinates is a full matrix. For cases where the coordinates are aligned, you should use the derived class, \mbox{\hyperlink{classoomph_1_1DiagQHermiteElement}{Diag\+Q\+Hermite\+Element}}, which uses a simplified mapping that makes the evaluation of derivatives of the shape functions much cheaper.
