\doxysubsection{Detailed Description}
Base class for \doxylink{classoomph_1_1MacroElement}{Macro\+Element} s that are used during mesh refinement in domains with curvlinear and/or time-\/dependent boundaries; see the description of the \doxylink{classoomph_1_1Domain}{Domain} class for more details. 

A macro element provides a parametrisation of a sub-\/domain by providing a mapping between a set of local coordinates $ {\bf S} $ and global coordinates $ {\bf r} ({\bf S}) $ . This must be implemented in the function
