\doxysubsection{Detailed Description}
\subsubsection*{template$<$class ELEMENT$>$\newline
class oomph\+::\+Tube\+Mesh$<$ ELEMENT $>$}
3D tube mesh class. The domain is specified by the \mbox{\hyperlink{classoomph_1_1GeomObject}{Geom\+Object}} that identifies the entire volume. Non-\/refineable base version! 

The mesh boundaries are numbered as follows\+:
\begin{DoxyItemize}
\item Boundary 0\+: \char`\"{}\+Inflow\char`\"{} cross section; located along the line parametrised by $ \xi_0 = \xi_0^{lo} $.
\item Boundary 1\+: The outer wall, represetned by $\xi_2 = 1$.
\item Boundary 2\+: The out flow, represented by $\xi_0 = \xi_0^{hi}$. 
\end{DoxyItemize}
