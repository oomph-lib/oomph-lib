//////////////////////////////////////////////////////////////////////// ////////////////////////////////////////////////////////////////////////

QGeneralisedAdvectionDiffusionElement elements are linear/quadrilateral/brick-shaped Advection Diffusion elements with isoparametric interpolation for the function.
