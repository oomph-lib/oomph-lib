template<unsigned DIM, unsigned NNODE_1D>
class oomph::QLinearWaveElement< DIM, NNODE_1D >

//////////////////////////////////////////////////////////////////////// ////////////////////////////////////////////////////////////////////////

QLinearWaveElement elements are linear/quadrilateral/brick-shaped LinearWave elements with isoparametric interpolation for the function.

Empty, just establishes the template parameters
