48  /// Refineable version of QElement<1,NNODE_1D>.
49  ///
50  /// Refinement is performed by binary tree procedures. When the element
51  /// is subdivided, the geometry of its sons is established by calls to
52  /// their father's \c get_x(...) function which refers to:
53  /// - the father element's geometric FE mapping (this is the default)
54  /// or
55  /// - a MacroElement's MacroElement::macro_map (if the pointer to the
56  /// macro element is non-NULL)
57  ///
58  /// The class provides a generic RefineableQElement<1>::build() function
59  /// which deals with generic isoparametric QElements in which all values
60  /// are associated with nodes. The RefineableQElement<1>::further_build()
61  /// function provides an interface for any element-specific non-generic
62  /// build operations.
63  //=======================================================================
