Detailed Description

/////////////////////////////////////////////////////////////////////////// ///////////////////////////////////////////////////////////////////////////

Crouzeix_Raviart elements are Navier–Stokes elements with quadratic interpolation for velocities and positions, but a discontinuous linear pressure interpolation
