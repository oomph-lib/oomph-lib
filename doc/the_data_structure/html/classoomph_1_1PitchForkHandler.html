Detailed Description

A class that is used to assemble the augmented system that defines a pitchfork (symmetry-breaking) bifurcation. The "standard" problem must be a function of a global parameter and a solution is , where are the unknowns in the problem. A pitchfork bifurcation may be specified by the augmented system of size .

In the abovem is the usual Jacobian matrix, and is a constant vector that is chosen to ensure that the null vector, , is not trivial. Here is a slack variable that is used to enforce the constraint — the inner product of the solution with the chosen symmetry vector is zero. At the bifurcation should be very close to zero. Note that this formulation means that any odd symmetry in the problem must be about zero. Moreover, we use the dot product of two vectors to calculate the inner product, rather than an integral over the domain and so the mesh must be symmetric in the appropriate directions.
