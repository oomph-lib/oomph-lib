Detailed Description

An Preconditioner class using the suite of Hypre preconditioners to allow 

 BoomerAMG (Algebraic Multi Grid),
 Euclid (ILU) or 
 ParaSails (Approximate inverse)

to be used for Preconditioner::preconditioner_solve(...). By default BoomerAMG is used.

This interface allows HyprePreconditioner to be used as a Preconditioner object for oomph-lib's own IterativeLinearSolver class. Note: matrix_pt must point to an object of type CRDoubleMatrix or DistributedCRDoubleMatrix.

As far as the Hypre solvers are concerned, the oomph-lib matrix is no longer required and could be deleted to save memory. However, this will not be the expected behaviour and therefore needs to be requested explicitly by the user by calling the enable_delete_matrix() function.
