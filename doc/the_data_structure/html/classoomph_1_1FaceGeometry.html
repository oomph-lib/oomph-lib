Detailed Description

template<class ELEMENT>
class oomph::FaceGeometry< ELEMENT >

FaceGeometry class definition: This policy class is used to allow construction of face elements that solve arbitrary equations without having to tamper with the corresponding "bulk" elements. The geometrical information for the face element must be specified by each "bulk" element using an explicit specialisation of this class.
