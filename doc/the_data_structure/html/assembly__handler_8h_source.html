449 /// system of size \f$ 2N+1 \f$
450 /// \f[ R(u,\lambda) = 0, \f]
451 /// \f[ Jy = 0, \f]
452 /// \f[\phi\cdot y = 1. \f]
453 /// In the above \f$ J \f$ is the usual Jacobian matrix, \f$ dR/du \f$, and
