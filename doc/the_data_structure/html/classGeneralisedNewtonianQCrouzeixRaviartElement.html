Detailed Description

template<unsigned DIM>
class GeneralisedNewtonianQCrouzeixRaviartElement< DIM >

/////////////////////////////////////////////////////////////////////////// ///////////////////////////////////////////////////////////////////////////

Crouzeix_Raviart elements are Navier–Stokes elements with quadratic interpolation for velocities and positions, but a discontinuous linear pressure interpolation. They can be used within oomph-lib's block preconditioning framework.
