51 /// Base class for MacroElement s that are used during mesh refinement
52 /// in domains with curvlinear and/or time-dependent
53 /// boundaries; see the description of the Domain class for more
54 /// details.
55 ///
56 /// A macro element provides a parametrisation of a sub-domain
57 /// by providing a mapping between a set of local coordinates
58 /// \f$ {\bf S} \f$ and global coordinates \f$ {\bf r} ({\bf S}) \f$ . This
59 /// must be implemented in the function
60 ///
61 /// \code MacroElement::macro_map(...) \endcode
62 ///
63 /// The time-dependent version of the macro map returns the mapping
64 /// from local to global coordinates: \f$ {\bf r} (t,{\bf S}) \f$
65 /// where t is the discrete timelevel: t=0: current time; t>0:
66 /// previous timestep.
67 ///
68 /// The MacroElement s establish the current (and previous) domain
69 /// shape via member function pointers to the Domain 's
70 /// \code Domain::macro_element_boundary(...) \endcode
71 /// member function.
72 //=================================================================
