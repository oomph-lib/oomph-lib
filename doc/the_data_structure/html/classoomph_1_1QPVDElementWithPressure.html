oomph::QPVDElementWithPressure< DIM > Class Template Reference

An Element that solves the equations of solid mechanics, using the principle of virtual displacements, with quadratic interpolation for the positions and a discontinuous linear solid pressure. This is analogous to the QCrouzeixRaviartElement element for fluids. More...

Detailed Description

template<unsigned DIM>
class oomph::QPVDElementWithPressure< DIM >

An Element that solves the equations of solid mechanics, using the principle of virtual displacements, with quadratic interpolation for the positions and a discontinuous linear solid pressure. This is analogous to the QCrouzeixRaviartElement element for fluids.
