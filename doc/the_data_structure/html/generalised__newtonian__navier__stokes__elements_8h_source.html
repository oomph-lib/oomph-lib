106 /// A class for elements that solve the cartesian Navier--Stokes equations,
107 /// templated by the dimension DIM.
108 /// This contains the generic maths -- any concrete implementation must
109 /// be derived from this.
110 ///
111 /// We're solving:
112 ///
113 /// \f$ { Re \left( St \frac{\partial u_i}{\partial t} +
114 /// (u_j - u_j^{M}) \frac{\partial u_i}{\partial x_j} \right) =
115 /// - \frac{\partial p}{\partial x_i} - R_\rho B_i(x_j) -
116 /// \frac{Re}{Fr} G_i +
117 /// \frac{\partial }{\partial x_j} \left[ R_\mu \left(
118 /// \frac{\partial u_i}{\partial x_j} +
119 /// \frac{\partial u_j}{\partial x_i} \right) \right] } \f$
120 ///
121 /// and
122 ///
123 /// \f$ { \frac{\partial u_i}{\partial x_i} = Q } \f$
124 ///
125 /// We also provide all functions required to use this element
126 /// in FSI problems, by deriving it from the FSIFluidElement base
127 /// class.
128 //======================================================================
