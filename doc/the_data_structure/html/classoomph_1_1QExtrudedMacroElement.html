Detailed Description

template<int DIM>
class oomph::QExtrudedMacroElement< DIM >

QExtrudedMacroElement

QExtrudedMacroElements are ExtrudedMacroElements with linear/quadrilateral/hexahedral shape. This class is empty and simply establishes the dimension as the template parameter.
