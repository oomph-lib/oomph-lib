45 /// If required, split the leaf element and create its sons
46 ///
47 /// Split is performed if
48 ///
49 /// object_pt()->to_be_refined()
50 ///
51 /// returns true.
52 ///
53 /// If this is the case, then we execute
54 ///
55 /// object_pt()->split(new_elements_pt)
56 ///
57 /// to create the sons. Pointers to the son objects are then inserted
58 /// into the son pointers of the present element. This turns the
59 /// present element into a "grey" (=non-leaf) node.
60 ///
61 //=================================================================
