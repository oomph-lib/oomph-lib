28 \section theory Theory
29 Consider a three-dimensional, axisymmetric body (of density \f$ \rho
30 \f$, Young's modulus \f$ E \f$, and Poisson's ratio \f$ \nu \f$),
31 occupying the region \f$ D \f$
32 whose boundary is \f$ \partial D \f$. Using cylindrical coordinates \f$
33 (r^*,\theta,z^*) \f$, the equations of linear elasticity can be written as
34 \f[
35 \nabla^* \cdot \bm{\tau}^* + \rho \mathbf{F}^* = \rho\frac{\partial^2\mathbf{u}^*}
36 {\partial {t^*}^2},
37 \f]
38 where \f$\nabla^* = \left(\frac{\partial}{\partial r^*},
39 \frac{1}{r^*}\frac{\partial}
40 {\partial \theta},\frac{\partial}{\partial z^*}\right)\f$,
41 \f$ \bm{\tau}^*(r^*,z^*,t^*) \f$ is the stress tensor,
42 \f$\mathbf{F}^*(r^*,z^*,t^*)\f$ is the body force and
43 \f$ \mathbf{u^*}(r^*,z^*,t^*) \f$ is the displacement field.
44 
45 Note that, despite the fact that none of the above physical quantities depend
46 on the azimuthal angle \f$ \theta \f$, each can have a non-zero \f$ \theta \f$
47 component. Also note that variables written with a superscript asterisk are
48 dimensional, and their non-dimensional counterparts will be written without an
49 asterisk. (The coordinate \f$ \theta \f$ is, by definition, non-dimensional, so
50 it will always be written without an asterisk.)
51 
52 A boundary traction \f$ \hat{\bm{\tau}}^* \f$ and boundary displacement \f$
53 \hat{\mathbf{u}}^* \f$ are imposed along the boundaries \f$ \partial
54 D_\mathrm{n} \f$ and \f$ \partial D_\mathrm{d} \f$ respectively, where \f$
55 \partial D = \partial D_\mathrm{d} \cup \partial D_\mathrm{n} \f$ so that
56 \f[
57 \mathbf{u}^* = \hat{\mathbf{u}}^*\text{ on }\partial D_\mathrm{d},
58 \qquad
59 \bm{\tau}^*\cdot\mathbf{n} = \hat{\bm{\tau}}^*\text{ on }\partial
60 D_\mathrm{n},
61 \f]
62 where \f$ \mathbf{n} \f$ is the outer unit normal vector.
63 
64 The constitutive equations relating the stresses to the displacements are
65 \f[
66 \bm{\tau}^* =
67 \frac{E}{1+\nu}\left(\frac{\nu}{1-2\nu}\left(\nabla^*\cdot\mathbf{u}^*\right)
68 \mathbf{I} + \frac{1}{2}\left(\nabla^*\mathbf{u}^* +
69 \left(\nabla^*\mathbf{u}^*\right)^\mathrm{T}\right)\right),
70 \f]
71 where
72 \f$ \mathbf{I} \f$ is the identity tensor and superscript \f$ \mathrm{T} \f$
73 denotes the transpose. In cylindrical coordinates, the matrix representation of
74 the tensor \f$ \nabla^*\mathbf{u}^* \f$ is
75 \f[
76 \nabla^*\mathbf{u}^* =
77 \begin{pmatrix}
78 \dfrac{\strut \partial u_r^*}{\strut \partial r^*} &
79 -\dfrac{\strut u_\theta^*}{\strut r^*} & \dfrac{\strut \partial
80 u_r^*}{\strut \partial z^*}\\
81 \dfrac{\strut \partial u_\theta^*}{\strut \partial r^*} &
82 \dfrac{\strut u_r^*}{\strut r^*} & \dfrac{\strut \partial
83 u_\theta^*}{\strut \partial z^*}\\
84 \dfrac{\strut \partial u_z^*}{\strut \partial r^*} & 0
85 & \dfrac{\strut \partial u_z^*}{\strut \partial z^*}
86 \end{pmatrix}
87 \f]
88 and \f$ \nabla^*\cdot \mathbf{u}^* \f$ is equal to the trace
89 of this matrix.
90 
91 We non-dimensionalise the equations, using a problem specific reference
92 length, \f$ \mathcal{L} \f$, and a timescale \f$ \mathcal{T} \f$, and use
93 Young's modulus to non-dimensionalise the body force and the stress tensor:
94 \f[
95 \bm{\tau}^* = E \bm{\tau}, \qquad
96 r^* = \mathcal{L}r, \qquad
97 z^* = \mathcal{L}z
98 \f]
99 \f[
100 \mathbf{u}^* = \mathcal{L} \mathbf{u} \qquad
101 \mathbf{F}^* = \frac{E}{\rho \mathcal{L}}\mathbf{F},
102 \qquad
103 t^* = \mathcal{T}t.
104 \f]
105 
106 The non-dimensional form of the axisymmetric linear elasticity equations is
107 then given by
108 \f[
109 \nabla\cdot \bm{\tau} + \mathbf{F} = \Lambda^2\frac{\partial^2\mathbf{u}}
110 {\partial t^2},\qquad\qquad (1)
111 \f]
112 where \f$ \nabla = \left(\frac{\partial}{\partial r},
113 \frac{1}{r}\frac{\partial}
114 {\partial \theta},\frac{\partial}{\partial z}\right) \f$,
115 \f[
116 \bm{\tau} =
117 \frac{1}{1+\nu}\left(\frac{\nu}{1-2\nu}\left(\nabla\cdot\mathbf{u}\right)
118 \mathbf{I} + \frac{1}{2}\left(\nabla\mathbf{u} +
119 \left(\nabla\mathbf{u}\right)^\mathrm{T}\right)\right),\qquad\qquad
120 (2)
121 \f]
122 and the non-dimensional parameter
123 \f[
124 \Lambda = \frac{\mathcal{L}}{\mathcal{T}}\sqrt{\frac{\rho}{E}}
125 \f]
126 is the ratio of the elastic body's intrinsic timescale,
127 \f$ \mathcal{L} \sqrt{\frac{\rho}{E}} \f$, to the problem-specific
128 timescale, \f$ \mathcal{T} \f$, that we use for time-dependent problems.
129 The boundary conditions are
130 \f[
131 \mathbf{u} = \hat{\mathbf{u}}\text{ on } \partial D_\mathrm{d}\qquad
132 \bm{\tau}\cdot\mathbf{n} = \hat{\bm{\tau}}\text{ on } \partial D_\mathrm{n}.
133 \f]
134 We must also specify initial conditions:
135 \f[
136 \left.\mathbf{u}\right|_{t=t_0} = \mathbf{u}^0,\qquad
137 \left.\frac{\partial\mathbf{u}}{\partial t}\right|_{t=t_0} =
138 \mathbf{v}^0.\qquad\qquad (3)
139 \f]

144 \section element_types Implementation
145 
146 Within \c oomph-lib, the non-dimensional version of the axisymmetric linear
147 elasticity equations (1) combined with the constitutive
148 equations (2) are implemented in the \c
149 AxisymmetricLinearElasticityEquations class. This class implements the
150 equations in a way which is general with respect to the specific element
151 geometry. To obtain a fully functioning element class, we must combine the
152 equations class with a specific geometric element class, as discussed in the
153 <A HREF="../../../quick_guide/html/index.html"> (Not-So-)Quick Guide</A>.
154 For example, we will combine the \c AxisymmetricLinearElasticityEquations class
155 with \c QElement<2,3> elements, which are 9-node quadrilateral elements, in our
156 example problem. As usual, the mapping between local and global (Eulerian)
157 coordinates within an element is given by
158 \f[
159 x_i = \sum_{j=1}^{N^{(E)}} X_{ij}^{(E)}\psi_j,\qquad i = 1,2,
160 \f]
161 where \f$ x_1 = r, x_2 = z \f$; \f$ N^{(E)} \f$ is the number of nodes in the
162 element. \f$ X_{ij}^{(E)} \f$ is the \f$ i \f$-th global (Eulerian) coordinate
163 of the \f$ j \f$-th node in the element and the \f$ \psi_j \f$ are the
164 element's shape functions, which are specific to each type of geometric
165 element.
166 
167 We store the three components of the displacement vector as nodal data in the
168 order \f$ u_r, u_z, u_\theta \f$ and use the shape functions to interpolate the
169 displacements as
170 \f[
171 u_i = \sum_{j=1}^{N^{(E)}}U_{ij}^{(E)}\psi_{ij},\qquad i = 1,\dotsc,3,
172 \f]
173 where \f$ U_{ij} \f$ is the \f$ i \f$-th displacement component at the \f$ j
174 \f$-th node in the element, i.e., \f$ u_1=u_r, u_2=u_z,u_3=u_\theta \f$.
175 
176 The solution of time dependent problems requires the specification of a \c
177 TimeStepper that is capable of approximating second time derivatives. In the
178 example problem below we use the Newmark timestepper.

183 \section test The test problem
184 
185 As a test problem we consider forced oscillations of the circular cylinder shown
186 in the sketch below:
187 \image html figures/geom.gif "Azimuthal cross-section of the geometry. "
188 \image latex figures/geom.eps "Azimuthal cross-section of the geometry. " width=0.35\textwidth
189 
190 It is difficult to find non-trivial exact solutions of the governing equations
191 (1), (2), so we manufacture a
192 time-harmonic solution:
193 
194 \f[
195 \mathbf{u} =
196 \begin{pmatrix}
197 u_r\\
198 u_\theta\\
199 u_z
200 \end{pmatrix} =
201 \cos(t)
202 \begin{pmatrix}
203 r^3\cos(z)\\
204 r^3z^3\\
205 r^3\sin(z)
206 \end{pmatrix}.\qquad\qquad (4)
207 \f]
208 This is an exact solution if we set the body force to
209 \f[
210 \mathbf{F} =
211 \cos(t)
212 \begin{pmatrix}
213 -r\cos(z)\left\{\left(8+3r\right)\lambda +
214 \left(16 - r\left(r-3\right)\right)\mu + r^2\Lambda^2\right\}\\
215 -r\left\{8z^3\mu + r^2\left(z^3\Lambda^2 + 6\mu z\right)\right\}\\
216 r\sin(z)\left\{- 9\mu + 4r\left(\lambda + \mu\right) +
217 r^2\left(\lambda + 2\mu -
218 \Lambda^2\right)\right\}
219 \end{pmatrix},\qquad\qquad (5)
220 \f]
221 where \f$ \lambda = \nu/((1+\nu)(1-2\nu)) \f$ and \f$ \mu = 1/(2(1+\nu)) \f$ are the nondimensional Lam&eacute; parameters.
222 We impose the displacement along the boundaries \f$ r=r_\mathrm{max},
223 z=z_\mathrm{min}, z=z_\mathrm{max} \f$ according to (4),
224 and impose the traction
225 \f[
226 \hat{\bm{\tau}}_3 = \bm{\tau}(r_\mathrm{min},z,t) =
227 \cos(\omega t)
228 \begin{pmatrix}
229 -\cos(z)r_\mathrm{min}^2\left(6\mu +
230 \lambda\left(4+r_\mathrm{min}\right)\right)\\
231 -2\mu r_\mathrm{min}^2z^3\\
232 -\mu r_\mathrm{min}^2\sin(z)\left(3-r_\mathrm{min}\right)
233 \end{pmatrix},\qquad\qquad (6)
234 \f]
235 along the boundary \f$ r = r_\mathrm{min} \f$.
236 
237 The problem we are solving then consists of equations
238 (1), (2) along with the body
239 force, (5) and boundary traction
240 (6). The initial conditions for the problem
241 are the exact displacement, velocity (and acceleration; see below) according to
242 the solution
243 (4).

376 \section initial_data Initial data
377 The time integration in this problem is performed using the Newmark scheme
378 which, in addition to the standard initial conditions
379 (3), requires an initial value for the acceleration.
380 Since we will be solving a test case in which the exact solution is known, we
381 can use the exact solution to provide the complete set of initial data required.
382 For the details of the Newmark scheme, see the tutorial on the
383 <A HREF="../../../linear_wave/two_d_linear_wave/html/index.html">linear wave equation</A>.
384 
385 If we're doing an impulsive start, set the displacement, velocity and
386 acceleration to zero, and fill in the time history to be consistent with this.

419 \section comments Comments
420 - Given that we non-dimensionalised all stresses on Young's modulus it seems odd
421 that we provide the option to specify a non-dimensional Young's modulus via
422 the member function \c AxisymmetricLinearElasticity::youngs_modulus_pt().
423 The explanation for this is that this function specifies the ratio of the
424 material's actual Young's modulus to the Young's modulus used in the
425 non-dimensionalisation of the equations. The capability to specify such ratios
426 is important in problems where the elastic body is made of multiple materials
427 with different constitutive properties. If the body is made of a single,
428 homogeneous material, the specification of the non-dimensional Young's modulus
429 is not required -- it defaults to 1.0.
