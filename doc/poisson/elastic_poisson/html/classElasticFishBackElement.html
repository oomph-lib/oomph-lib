Detailed Description

Fish back as a GeneralisedElement: Circular ring whose position is given by.

The ring's vertical position is determined by "pseudo elasticity":

This simulates the case where the centre of the ring is mounted on an elastic spring of stiffness and loaded by the force The "load" is specified by the Data object load_pt().
