3 \mainpage Example codes and Tutorials
4 
5 This document provides a complete list of the example codes that are
6 distributed with the \c oomph-lib library. For each
7 code we give a brief description of the problem solved and
8 provide a link to the detailed documentation. The
9 codes are listed in order of increasing complexity. The bullet-point
10 list in the right column lists the new \c oomph-lib features
11 that are introduced in the example. You may either work
12 through the examples one-by-one, treating the example codes and their
13 documentation as chapters in a self-study course, or use the list of
14 topics in the right column as a quick reference to example codes
15 that provide an introduction to a specific feature.
16 
17 You may also wish to consult the following documents:
18 - The <A HREF="../../intro/html/index.html">Introduction </A> provides
19  a "low tech" review of the theory of finite elements and presents a
20  "top-down" discussion of the method's implementation within \c oomph-lib.
21 - The document <A HREF="../../the_data_structure/html/index.html">
22  The Data Structure </A> provides a "bottom-up" discussion of
23  \c oomph-lib's data structure.
24 - The <A HREF="../../quick_guide/html/index.html">(Not-So-)Quick
25  Guide</A> provides a "quick" introduction on how to create new
26  instances of \c oomph-lib's fundamental objects: \c Problems,
27  \c Meshes, and \c Elements.
28 - The <A HREF="../../mpi/general_mpi/html/index.html">
29  tutorial discussing <code>oomph-lib</code>'s parallel processing
30  capabilities.</a>.
