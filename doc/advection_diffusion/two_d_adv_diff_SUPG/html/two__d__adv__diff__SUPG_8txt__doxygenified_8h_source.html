3 \mainpage Example problem: SUPG-stabilised solution of the 2D advection diffusion equation
4 
5 In this example we discuss the SUPG-stabilised solution of the 2D
6 advection-diffusion problem
7 
8 <CENTER>
9 <TABLE>
10 <TR>
11 <TD>
12 <CENTER>
13 <B>Two-dimensional advection-diffusion problem in a
14 rectangular domain</B>
15 </CENTER>
16 Solve
17 \f[
18 \mbox{Pe}\ \sum_{i=1}^2 w_i \left(x_1,x_2\right) \frac{\partial u}{\partial
19 x_i} = \sum_{i=1}^2 \frac{\partial^2u}{\partial x_i^2} + f(x_1,x_2),
20  \ \ \ \ \ \ \ \ \ \ (1)
21 \f]
22 in the rectangular domain \f$D =\left\{ (x_1,x_2) \in
23 [0,1] \times [0,2]\right\}\f$,
24 with Dirichlet boundary conditions
25 \f[
26 \left. u\right|_{\partial D}=u_0,
27 \ \ \ \ \ \ \ \ \ \ (2)
28 \f]
29 
30 where the \e Peclet \e number, \f$ \mbox{Pe} \f$ the boundary values,
31 \f$ u_0\f$, the source function \f$ f(x_1,x_2),\f$
32 and the components of the "wind" \f$ w_i(x_1,x_2) \ (i=1,2)\f$ are given.
33 </TD>
34 </TR>
35 </TABLE>
36 </CENTER>
37 
38 We set \f$ f(x_1,x_2)=0\f$ and assign the boundary conditions such that
39 \f[
40 u_0(x_1,x_2) = \tanh(1-\alpha(x_1 \tan\Phi - x_2)),
41 \ \ \ \ \ \ \ \ \ (3)
42 \f]
43 For large values of \f$ \alpha, \f$
44 this boundary data approaches a step, oriented at an angle
45 \f$\Phi\f$ against the \f$x_1-\f$axis.
46 
47 
48  In the computations we will impose the "wind"
49 \f[
50 \mathbf{w}\left(x_1,x_2 \right) =
51 \left(
52 \begin{array}{c}
53 \sin(6x_2) \\
54  \cos(6x_1)
55 \end{array}
56 \right),
57 \ \ \ \ \ \ \ \ \ (4)
58 \f]
59 illustrated in this vector plot:
60 
61 \image html wind.gif "Plot of the wind. "
62 \image latex wind.eps "Plot of the wind. " width=0.75\textwidth
63 
64 
65 The figures below show plots of the solution for
66 \f$\Phi=45^\circ, \ \alpha=50 \f$
67 and a Peclet number of \f$ Pe = 200\f$, with and without SUPG
68 stabilisation. The wire-mesh plot shows the solution computed
69 on a 10x10 mesh, the shaded surface represents the solution
70 obtained from an unstabilised solution on a 150x150 mesh.
71 Note how SUPG stabilisation "suppresses the wiggles" on the
72 relatively coarse mesh.
