Detailed Description

template<class ELEMENT>
class TwoMeshFluxAdvectionDiffusionProblem< ELEMENT >

2D AdvectionDiffusion problem on rectangular domain, discretised with 2D QAdvectionDiffusion elements. Flux boundary conditions are applied along boundary 1 (the boundary where x=L). The specific type of element is specified via the template parameter.
