3 \mainpage Example problem: Adaptive solution of the 2D advection diffusion equation with flux boundary conditions
4 
5 In this problem we will discuss the 2D advection-diffusion problem
6 with Neumann (flux) boundary conditions, using \c oomph-lib's
7 mesh adaptation routines.
8 
9 <CENTER>
10 <TABLE>
11 <TR>
12 <TD>
13 <CENTER>
14 <B>Two-dimensional advection-diffusion problem in a rectangular domain</B>
15 </CENTER>
16 Solve
17 \f[
18 \mbox{Pe}\ \sum_{i=1}^2 w_i \left(x_1,x_2 \right) \frac{\partial u}{\partial
19 x_i} = \sum_{i=1}^2 \frac{\partial^2u}{\partial x_i^2} + f(x_1,x_2),
20  \ \ \ \ \ \ \ \ \ \ (1)
21 \f]
22 in the rectangular domain \f$D =\left\{ (x_1,x_2) \in
23 [0,1] \times [0,2]\right\}\f$. We split the domain boundary \f$
24 \partial D \f$ into two parts so that
25 \f$ \partial D = \partial D_{Neumann} \cup \partial D_{Dirichlet} \f$,
26 where \f$ \partial D_{Neumann}
27 = \left\{ (x_1,x_2) | x_1=1, \ x_2\in [0,2] \right\} \f$.
28 On \f$ \partial D_{Dirichlet}\f$ we apply the Dirichlet boundary conditions
29 \f[
30 \left. u\right|_{\partial D_{Dirichlet}}=u_0,
31 \ \ \ \ \ \ \ \ \ \ (2)
32 \f]
33 where the function \f$ u_0 \f$ is given. On \f$ \partial D_{Neumann}\f$
34 we apply the Neumann conditions
35 \f[
36 \left. \frac{\partial u}{\partial n}\right|_{\partial D_{Neumann}} =
37 \left. \frac{\partial u}{\partial x_1}\right|_{\partial D_{Neumann}}
38 =g_0,
39 \ \ \ \ \ \ \ \ \ \ (3)
40 \f]
41 where the function \f$ g_0 \f$ is given.
42 </TD>
43 </TR>
44 </TABLE>
45 </CENTER>
46 
47 As always, we validate the code by choosing the boundary data and the
48 source functions such that
49 \f[
50 u(x_1,x_2) = \tanh(1-\alpha(x_1 \tan\Phi - x_2)),
51 \ \ \ \ \ \ \ \ \ (4)
52 \f]
53 is the exact solution of the problem.
54 The plot below shows the numerical solution for \f$\Phi=45^\circ \f$,
55 a Peclet number of \f$\mbox{Pe}= 200 \f$, and four
56 different values of the "steepness parameter",
57 \f$ \alpha = 0.2,\ 5,\ 10 \f$ and \f$ 15 \f$.
58 
59 \image html step_soln.gif "Animation of the adaptive solution for various values of the `steepness parameter'. "
60 \image latex step_soln.eps "Animation of the adaptive solution for various values of the `steepness parameter'. " width=0.75\textwidth
61 
62 As in the <A HREF="../../two_d_adv_diff_adapt/html/index.html">example
63 with Dirichlet boundary conditions</A>, the unforced case is a lot
64 more interesting. The plot below shows the result for a zero source function
65 \f$ f \equiv 0 \f$, Dirichlet boundary conditions determined from
66 the "exact solution" of the forced problem for \f$ \alpha = 15\f$,
67 and a prescribed flux of \f$ g_0 = -1 \f$ on \f$ \partial D_{Neumann}\f$.
68 
69 
70 \image html no_source.gif "Plot of the adaptive solution of the unforced problem. "
71 \image latex no_source.eps "Plot of the adaptive solution of the unforced problem. " width=0.75\textwidth
72 
73 Along \f$ \partial D_{Dirichlet}\f$, the value of \f$ u \f$ is enforced
74 by the Dirichlet boundary condition (2) and,
75 as in the <A HREF="../../two_d_adv_diff_adapt/html/index.html">
76 previous example</A>, the "wind" either sweeps this value into the
77 interior of the domain or creates a sharp boundary layer within
78 which the solution that is "swept" along from the interior adjusts itself
79 to the prescribed boundary value. Along \f$ \partial D_{Neumann}\f$, the
80 flux boundary condition (3) imposes the normal
81 derivative of the solution. This boundary condition is much
82 "softer" than the Dirichlet condition and does not create boundary
83 layers that are as sharp as the ones that develop on
84  \f$ \partial D_{Dirichlet}\f$.
