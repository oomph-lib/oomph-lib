\doxysubsection*{Licencing details for third-\/party libraries}

{\ttfamily oomph-\/lib} provides the option to install various third-\/party libraries as part of its overall build process. These are\+: 
\begin{DoxyItemize}
\item \href{https://github.com/OpenMathLib/OpenBLAS}{\texttt{ Open\+BLAS}} 
\item \href{https://github.com/boostorg/boost}{\texttt{ Boost}} 
\item \href{https://github.com/CGAL/cgal/}{\texttt{ CGAL}} 
\item \href{https://github.com/hypre-space/hypre/}{\texttt{ HYPRE}} 
\item \href{https://mumps-solver.org/index.php}{\texttt{ MUMPS}} (built via a fork of the \href{https://github.com/scivision/mumps}{\texttt{ Scivision MUMPS via CMake}} project) 
\item \href{https://github.com/KarypisLab/GKlib}{\texttt{ GKlib}} 
\item \href{https://github.com/KarypisLab/METIS}{\texttt{ METIS}} 
\item \href{https://github.com/KarypisLab/ParMETIS}{\texttt{ Par\+METIS}} 
\item \href{https://github.com/xiaoyeli/superlu}{\texttt{ Super\+LU}} 
\item \href{https://github.com/xiaoyeli/superlu_dist}{\texttt{ Super\+LU\+\_\+dist}} 
\item \href{https://github.com/trilinos/Trilinos/}{\texttt{ Trilinos}} 
\end{DoxyItemize}

Our build scripts download the relevant source automatically from the relevant Git\+Hub repositories where the libraries\textquotesingle{} licencing information can be found.
