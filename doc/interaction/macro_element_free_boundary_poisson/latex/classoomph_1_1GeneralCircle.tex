\doxylink{classoomph_1_1GeneralCircle}{General\+Circle} in 2D space. 

\[ x = X_c + R \cos(\zeta)  \] \[ y = Y_c + R \sin(\zeta)  \] The three parameters $ X_c, Y_c $ and $ R $ are represented by Data and can therefore be unknowns in the problem.
