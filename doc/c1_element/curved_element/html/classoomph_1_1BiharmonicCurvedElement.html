Detailed Description

template<unsigned DIM, unsigned NNODE_1D>
class oomph::BiharmonicCurvedElement< DIM, NNODE_1D >

BiharmonicCurvedElement elements are a subparametric scheme with a Lagrange interpolation for approximating the geometry and the C1-functions for approximating variables.
