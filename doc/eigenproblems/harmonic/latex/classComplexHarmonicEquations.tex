\doxysubsection{Detailed Description}
A class for all elements that solve the eigenvalue problem. 

\[  \frac{\partial w}{\partial x}  = \lambda u \] \[ \frac{\partial u}{\partial x} = (\lambda - \mu) w \] This class contains the generic maths. Shape functions, geometric mapping etc. must get implemented in derived class.
