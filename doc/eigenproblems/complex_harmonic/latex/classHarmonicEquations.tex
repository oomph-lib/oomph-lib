\doxysubsection{Detailed Description}
A class for all elements that solve the simple one-\/dimensional eigenvalue problem. 

\[ \frac{\partial^2 u}{\partial x_i^2}  + \lambda u = 0 \] These elements are very closely related to the Poisson elements and could inherit from them. They are here developed from scratch for pedagogical purposes. This class contains the generic maths. Shape functions, geometric mapping etc. must get implemented in derived class.
