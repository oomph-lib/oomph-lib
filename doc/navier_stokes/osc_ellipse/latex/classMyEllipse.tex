\doxysubsection{Detailed Description}
Oscillating ellipse. 

\[ x = (A + \widehat{A} \cos(2\pi t/T)) \cos(\xi) \] \[ y = \frac{\sin(\xi)}{A + \widehat{A} \cos(2\pi t/T)} \] Note that cross-\/sectional area is conserved.
